Student Feedback Management System
A modern, full-stack web application for managing student feedback on courses. Built with Node.js/Express backend and vanilla JavaScript frontend.
Features

✅ Submit course feedback with star ratings
✅ View all feedbacks with filtering options
✅ Real-time statistics and analytics per course
✅ Responsive design for mobile and desktop
✅ Modern, clean UI with smooth animations
✅ Admin capabilities to manage feedbacks

Tech Stack

Backend: Node.js, Express.js
Frontend: HTML5, CSS3, Vanilla JavaScript
Deployment: Render-ready
